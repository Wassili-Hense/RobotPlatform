### I2C protocol
Slave address: 0x14

### Read from slave (fixed packet)

Slave returns one fixed-size packet of 6 bytes on every read transaction.

Packet bytes:
- Byte0 = word0[7:0]
  - bit0 - Button ON
  - bit1 - Button Fire
  - bit2 - Button UP
  - bit3 - Button DOWN
  - bit4 - Button BACK
  - bit5 - Button OK
  - bit6 - Button LUP
  - bit7 - Button LDN
- Byte1 = word0[15:8]
  - bit0 - Button RUP
  - bit1 - Button RDN
  - bit2..3 - reserved, 0
  - bit4 - USB connected (1 -> connected, 0 -> not connected)
  - bit5 - reserved, 0
  - bit6 - LCD backlight state (1 = on, 0 = off)
  - bit7 - LCD busy / flow control (1 = master must stop sending LCD commands, 0 = master may send LCD commands)
- Byte2 = ADC X value[7:0]
- Byte3
  - bit0..3 - ADC X value[11:8]
  - bit4..6 - reserved, 0
  - bit7 ADC X channel changed
- Byte4 = ADC Y value[7:0]
- Byte5
  - bit0..3 - ADC Y value[11:8]
  - bit4..6 - reserved, 0
  - bit7 ADC Y channel changed

Buttons bit = 1 means button pressed

### Write to slave

One command per I2C write transaction.
Maximum write length: 32 bytes.
Multibyte parameters are little-endian.

- BacklightTimeout(uint32 timeout_ms)
  - format: [0x03][t0][t1][t2][t3]

- BacklightBrightness(level)
  - level: 0..127
  - format: [0x04][level]

- Tone(divider, delay_ms)
  - divider = 1000000 / Hz
  - format: [0x07][divider_lo][divider_hi][delay_lo][delay_hi]

- LCD_Clear(color)
  - color: RGB565
  - format: [0x10][color_lo][color_hi]
  - LCD service time in slave: about 48.818 ms

- LCD_FillRect(x, y, w, h, color)
  - format: [0x11][x][y][w][h][color_lo][color_hi]
  - LCD service time in slave: about 22 us * h + 4 us * w * h

- LCD_DrawMarker(x, y, idx, color)
  - x, y - marker center
  - idx  - marker index (size is taken from internal table)
  - format: [0x12][x][y][idx][color_lo][color_hi]
  - LCD service time in slave: about 0.278 ms

- LCD_DrawText(x, y, color, text)
  - color: RGB565
  - text must be zero-terminated
  - text max length is limited by the LCD driver queue item storage
  - background color is global (see LCD_SetBackgroundColor)
  - format: [0x13][x][y][color_lo][color_hi][text...][0]
  - LCD service time in slave: about 0.342 ms per character

- LCD_SetBackgroundColor(color)
  - color: RGB565
  - format: [0x14][color_lo][color_hi]
  - sets the global background color used by LCD_DrawText() and LCD_DrawMarker()

- LCD_Indicator(index, state)
  - index: [0 - RF, 1 - USB]
  - state: [0 - off, 1 - on]
  - format: [0x20][index][state]
  - LCD service time in slave: about 0.166 ms

- LCD_DrawProgressBar(index, value)
  - index:
    - 0 - left vertical
    - 1 - top left
    - 2 - top right
  - value: 0..70
  - format: [0x21][index][value]
  - LCD service time in slave: about 4.760 ms

Note:
- progress bar index 3 (right vertical) is used internally by firmware in App_ProcessAdc() for battery indication and is not exposed via I2C write command.
