Техзадание
1. Платформа

Arduino / ESP32-D0WD-V3
I2C master:

SDA = 21
SCL = 22
адрес slave: 0x14


Экран подключён к I2C slave:

RGB565
charset[32..126]
шрифт 10x7
доступная область: x=10..149, y=10..79

2. Опрос и вывод в Serial

Период опроса I2C: 5 мс
По каждому принятому I2C-пакету печатать строку:

length([0]&0x0F) status([1]) X Y button


Для length==1 печатать только при изменении status
Формат:

length→ %2u
status →%2x
X, Y → [-1..1], формат %.3f
button → имя кнопки без BUTTON_

Если нажато несколько кнопок — выводить кнопку с меньшим индексом

Лог всех исходящих I2C-команд:

формат TX 20 01 01

3. LCD flow control

После получения STATUS:

s_lcdSendAllowed = (LCD busy == 0)


Любая LCD_* команда может отправляться только если

s_lcdSendAllowed == true


После любой успешно отправленной LCD_* команды:

s_lcdSendAllowed = false

приоритет операций
высокий: Tone, LCD_DrawMarker
средний: остальные
низкий: LCD_DrawProgressBar, LCD_Indicator

4. USB / Serial
При обработке STATUS:

если Usb connected изменился на 0:

Serial.end()
LCD_Indicator(1, 0)

если Usb connected изменился на 1:

Serial.begin(115200)
LCD_Indicator(1, 1)

5. Джойстик и маркер
Обычный режим

Рисовать положение джойстика как маркер:

LCD_DrawMarker(idx = 3, color = белый)
область: x=40..119, y=10..79
Старый маркер стирать чёрным
если LCD backlight state == off, стереть и не обновлять

Калибровка

Cal center

LCD_DrawMarker(idx = 5, color = зелёный)
старые маркеры не стирать
OK — сохранить
BACK — не сохранять
Маштабировать поле вывода. w=h=2*max(предыдужие x.cMax - x.CMin, y.cMax - y.cMin, 16), center - (x.cMax + x.CMin)/2, (y.cMax + y.cMin) / 2
Ширина dead zone не менее 30.

Cal edge

LCD_DrawMarker(idx = 3, color = синий)
старые маркеры не стирать
OK — сохранить
BACK — не сохранять


6. Калибровка осей
Для каждой оси хранить 4 значения:

[min, cMin, cMax, max]

Интерпретация:

min  → -1
cMin → 0
cMax → 0
max  → 1

Значения сохранять в Preferences.

7. Меню
Вход

Кнопка OK открывает меню

Пункты

Cal center
Cal edge

Управление в меню

UP / DOWN — выбор пункта
OK — вход / подтверждение
BACK — назад / выход
LUP / LDN — регулировка яркости

Текст калибровки

Текст режимов калибровки рисовать справа, вне зоны маркера, вертикально(через /n), Слева - cancel, справа - accept


8. Яркость и подсветка
Яркость

Хранить в 11 логарифмических шагах в диапазоне[1..127]
Сохранять в Preferences

Поведение

при входе в меню:

SendBacklightBrightness(s_backlightLevel)


при выходе из меню:

SendBacklightBrightness(max(s_backlightLevel / 2, 1))


при изменении яркости в меню:

обновлять только поле яркости, без полной перерисовки меню



Timeout подсветки

пока меню активно:

слать SendBacklightTimeout(15000) каждые 10 секунд



После старта

загрузить яркость и калибровку из Preferences
яркость отправлять только после первого успешно принятого STATUS
отправлять max(s_backlightLevel / 2, 1)


9. Выход из меню по не активности

Если 60 секунд нет активности — выйти из меню
Активность = только изменение состояния кнопок
Джойстик на таймер неактивности не влияет


10. Команды из Serial
Поддержать команды:

ANN → DrawProgressBar(0, value)
BNN → DrawProgressBar(1, value)
CNN → DrawProgressBar(2, value)
DN  → LCD_Indicator(0, value)
THz,Ms → Tone(1000000/Hz, Ms)
