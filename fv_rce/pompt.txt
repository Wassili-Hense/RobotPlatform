Экран(подключен к I2C slave): R5G6B5, charset[32 .. 126],Font 10x7, доступная область x0=10, y0=10, x1=139, y1=79. 

читает с I2C slave, период опроса 5 мс.
Печать в Serial по приходу пакета из I2C, кроме пакета STATUS, его выводить только при изменении value. данные выводятся без заголовков разделённые " ":
- индекс из пришедшей комманды (%1x)
- статус (%d) 
- X и Y. [-1 .. 1] в формате (%.3f) 
- имя нажатой кнопки(без BUTTON_), если несколько, то с меньшим индексом 

вывод в Serial всего, что отправляется в I2C. Лог отправки имеет вид: TX 20 01 01

при получении STATUS{
  s_lcdSendAllowed = LCD busy == 0;

  если Usb connected изменился на 0{
    вызывается Serial.end()
    LCD_Indicator(1, 0)
  }
  если изменился на 1 {
    Serial.begin(115200)
    LCD_Indicator(1, 1)
  }
}
после любой успешно отправленной LCD-команды: s_lcdSendAllowed = false;
LCD_ команды можно посылать только s_lcdSendAllowed == true.

рисует положение джойстика как белый маркер FillCircle(r=3) в области x=10..79, y=10..79;
старый маркер стирает чёрным;
Отрисовка маркера имеет приоритет перед другими LCD_ командами. 

по OK открывает меню;
пункты меню:
 * Cal center — калибровка центра, до OK - сохранить изменения/BACK - не сохранять, маркер зелёный, старые не стираются;
 * Cal edge — калибровка края, до OK/BACK, маркер синий, старые не стираются;

Для каждой оси джойстика хранится 4 переменных: [min, cMin, cMax, max] с переводом [-1, 0, 0, 1] 

пока меню активно:

UP/DOWN — выбор пункта;
OK — вход/подтверждение;
BACK — назад/выход;
LUP/LDN — регулировка яркости;

 - перенеси текст калибровки вправо.
 - при входе в меню SendBacklightBrightness(s_backlightLevel) при выходе SendBacklightBrightness(мах(s_backlightLevel / 2, 1)).
 - при нахождении в меню слать SendBacklightTimeout(15000) каждые 10 секунд.
 - при не активности(активность - изменение состояния кнопок)дольше 60 секунд выход из меню.  
 - настройки джойстика и яркости сохраняются в Preferences;
 - яркость после старта отправляется после первого успешно принятого STATUS согласно меню/не меню;

команды из Serial:

ANN → DrawProgressBar(0, value)
BNN → DrawProgressBar(1, value)
CNN → DrawProgressBar(2, value)
DN  → LCD_Indicator(0, value)
THz,Ms → Tone(1000000/Hz, Ms)
